# Included workflows

The repository demonstrates two common ways to organise a thesis:

1. **Direct use:** the runnable project at the repository root keeps chapter
   files inside `Chapters/` and lists them in `_quarto.yml`.
2. **External manuscripts:** `external-manuscripts/` shows how a wrapper
   chapter can include manuscript sections stored elsewhere in a larger
   research repository.

The external-manuscript files are a structural example. Copy the wrapper and
manuscript layout into your own repository and adjust every relative path to
match your folders.
